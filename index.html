Cheat Sheet

Position & Display

Property: Position

Value Description
Static Default position where the elements follow the flow of html
Relative Element is relative to it's initial position and can be off set with properties like: Top, right, etc.
Absolute Element is completly removed from the html flow and It's position relative to the closest ancestor element. It can be off set with direction properties.
Fixed Element is position relative to the viewport and it will keep its position regarless of scrolling. Direction properties can off set its postion.
Sticky Element is positioned based on the user's scroll position. Sticky elements toggle between relative and fixed position. One must establish a direction propertie for the element to stick.

Property: Display

Value Description
In-line Element is able to share the same line with other elemnts. width and height will have no effect.
Block Element is display as a block and it will take the full width of the parent container.
In-line Block Element will be displayed as an in-line but will be able to share horizontal space with other elements. Widht and and height can be defined.
